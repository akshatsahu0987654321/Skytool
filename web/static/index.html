👋 Welcome

        
Sky Tools is a batch CGPT signup + 2FA pipeline with a local web UI. This guide walks you through every tab, mail mode and the most common pitfalls. Stuck? Tap the 💬 @skywalker1_1 badge anywhere — that opens our Telegram support.

      


      
      

        
🚀 Quick Start — your first signup in 3 steps

        
    
          
  1. Open the Reg tab. Pick a Mail Mode (we recommend icloud_v3 — see the Mail Modes section). Paste your combos into the textarea — one per line.
    2. 
          
  2. Choose Reg Mode and Mode. Hybrid is the safest balance of speed and anti-detect. The Mode dropdown in the header controls how many jobs run in parallel (Single, Multi 5, Multi 30…).
    3. 
          
  3. Press Start. Watch live progress in the Jobs card, raw logs in the Log card, and successful credentials drop into the Free / Plus output panes — ready to copy.
    4. 
        

      


      
      

        
🗂️ Tabs at a glance

        
    
          
  • Reg — create new CGPT accounts in bulk, with 2FA auto-enabled.
    • 
          
  • Get Session — paste email|password|totp_secret combos and harvest fresh session.json + access tokens.
    • 
          
  • UPI QR — turn an account into a payable UPI QR (India CGPT Plus checkout). Optional Telegram delivery of the QR + credentials.
    • 
          
  • Settings — proxy pool, Telegram bot, Cloudflare quick tunnel and a few more knobs.
    • 
        

      


      
      

        
📧 Mail Modes — pick the right one

        
    
          
  • icloud_v3 (recommended) — one entry per line: email|api_url. Best detection bypass + fast OTP. Need iCloud mail? Contact @skywalker1_1.
    • 
          
  • outlook — Outlook combos: email|password|refresh_token|client_id.
    • 
          
  • worker — iCloud email via Worker OTP. Format: just user@icloud.com. Need mail? Contact @skywalker1_1.
    • 
          
  • gmail_advanced — Gmail through an OTP-fetch API; supports api_url or email|api_url with mail_status pre-check.
    • 
        

      


      
      

        
⚙️ Reg Modes — Browser vs Hybrid

        
    
          
  • Browser — full Camoufox flow. Slowest but easiest to verify visually. Use Headless OFF + Debug ON when developing.
    • 
          
  • Hybrid (recommended) — Camoufox handles auth + sentinel tokens, pure HTTP runs the rest. Fast and stealthy at the same time.
    • 
        

        
Headless toggle hides the browser window. Debug toggle keeps the browser open after the job finishes — useful for inspecting failed runs.

      


      
      

        
💰 Get the cheap CGPT Plus price (40–60% off)

        
CGPT Plus uses purchasing-power-parity pricing. To get the best deal, the login traffic must exit from Vietnam (VN) or Japan (JP).

        
    
          
  • ✅ Easiest: rent a JP/VN VPS (Vultr Tokyo ~$5/month) and run the tool there — no proxy needed.
    • 
          
  • ✅ Alternative: install a JP/VN VPN on the device running the tool. ProtonVPN free tier includes a free Japan node.
    • 
          
  • ✅ Production: configure proxy pool in Settings → Proxies with VN/JP residential proxies.
    • 
          
  • ⛔ Running from US/EU without a JP/VN exit IP → Plus price is 2–3× higher and UPI/GoPay may be geo-blocked.
    • 
        

      


      
      

        
💳 UPI QR (India payment)

        
UPI lets Indian CGPT accounts pay for Plus instantly via PhonePe / GPay / Paytm.

        
    
          
  1. Open the UPI QR tab and paste email|password|totp_secret combos (one per line).
    2. 
          
  2. Tune Approve retries, Restart threshold, Re-login cycles, and Block-streak if you hit rate limits.
    3. 
          
  3. Optionally enable Send Telegram — each successful job DMs the QR + credentials to your configured chat.
    4. 
          
  4. Press Start. QR pops up in a modal as soon as the checkout is ready.
    5. 
        

        
For high-volume UPI generation, look at the standalone rust_upi_bot/ — it runs as a Telegram bot with a worker pool and per-user limits.

      


      
      

        
🔧 Settings deep dive

        
    
          
  • Proxies — login-flow strategy (anti409 for batch password combos, legacy for OTP / passwordless landing) and the proxy pool. Supports host:port:user:pass, full URL, and template {SID} for sticky sessions.
    • 
          
  • Telegram — bot token from @BotFather, chat ID from @userinfobot. The “Send Telegram” toggle on UPI QR delivers the QR plus a spoiler-message containing email|pass|secret.
    • 
          
  • Tunnel — one-click Cloudflare Quick Tunnel publishes the local UI at a random *.trycloudflare.com URL. Auto-downloads the cloudflared binary the first time. No Cloudflare account required.
    • 
        

      


      
      

        
🩹 Troubleshooting cheat-sheet

        
    
          
  • Captcha / Turnstile keeps failing → switch to a residential proxy; datacenter IPs get banned aggressively (esp. India).
    • 
          
  • UPI confirm rejected → Stripe needs 3 JS-runtime tokens. Use the bundled Rust UPI bot or keep retries high.
    • 
          
  • Plus checkout shows the expensive price → your exit IP isn't JP/VN. See the “Get the cheap price” section above.
    • 
          
  • Job stuck on “running” → check the Log card, the proxy pool status, and consider lowering Mode / concurrency.
    • 
          
  • Web UI blank after refresh → make sure the URL still carries ?token=…, or open it via the auto-injected link.

👋 Welcome

        
SKY is a batch CGPT signup & payment automation tool. It runs locally on your machine and exposes a web UI to register accounts, fetch sessions, and pay UPI in bulk — with realtime progress for every job.

        
💡 Pricing tip: CGPT Plus is 40–60% cheaper in Vietnam (VN) and Japan (JP) due to PPP pricing. If you don't have a VN/JP login proxy, install a VN or JP VPN on the device running this tool to inherit the cheap region.

      


      

        
⚡ Quick Start (3 steps)

        
    
          
  1. Pick a Mail Mode (top of Reg tab) — iCloud v3 is recommended. Don't have iCloud mails? Contact @skywalker1_1 on Telegram to buy.
    2. 
          
  2. Paste combos — one entry per line, following the placeholder format shown in the textarea.
    3. 
          
  3. Pick Mode (top-right: Single / Multi 2/3/5/10/...) and press Run. Watch jobs stream in the Jobs panel.
    4. 
        

      


      

        
🗂️ Tab Guide

        

          
Reg
Register fresh CGPT accounts in batch. Outputs email|password|2fa_secret.

          
Get Session
Take existing combos and extract usable session JSON (access_token + cookies) for downstream tools.

          
UPI QR
Run the CGPT Plus UPI checkout flow for India accounts and render a payable QR code. Optional Telegram delivery.

          
Settings
Configure proxy pool, Telegram bot, Cloudflare quick tunnel, and global flags (login flow, fingerprint).

        

      


      

        
📧 Mail Modes (Reg tab)

        

          
iCloud v3 recommended
Each line: email|api_url (Worker v2 readmail URL). Most reliable OTP delivery. Contact @skywalker1_1 to buy iCloud mails.

          
Outlook
Format email|password|refresh_token|client_id. Useful if you have an Outlook pool already.

          
iCloud Worker
Plain iCloud emails, one per line — OTP fetched via shared Worker. Need iCloud mails? @skywalker1_1.

          
Gmail (advanced)
Pre-checked Gmail addresses via custom OTP API. Each line: api_url or email|api_url.

        

      


      

        
🌐 Reg Modes

        

          
Browser (anti-detect)
Full Camoufox (Firefox-shaped) flow. Highest stealth, slowest. Best for difficult IPs.

          
Hybrid default
Pure-Python HTTP + Camoufox sentinel sidecar for token minting. Much faster than Browser, similar success rate on clean proxies.

        

        
💡 Use Headless toggle to hide browser windows. Enable Debug to keep them open when headed (for inspection).

      


      

        
🔌 Settings → Proxies

        
Paste one proxy per line. Three accepted formats:

        
    
          
  • host:port:user:pass
    •

    • 
          
  • socks5://user:pass@host:port
    • 
        

        
Login flow: anti409 = best for batch password combos (warm + sentinel verify, avoids HTTP 409). legacy = use for passwordless/OTP accounts when landing page is ambiguous.

🚇 Settings → Cloudflare Quick Tunnel

        
One click to expose your local UI as https://*.trycloudflare.com. No Cloudflare account required — a random URL is issued each time. The tool downloads the cloudflared binary automatically on first use (Win / Mac / Linux, x64 / arm64).

      


      

        
💰 UPI QR — How it Works

        
    
          
  1. Paste combos: email|password|totp_secret (India accounts).
    2. 
          
  2. Tool logs in, creates a CGPT Plus checkout, initializes Stripe, fetches elements, confirms with your UPI VPA, then polls approve until you scan the QR with PhonePe / GPay / Paytm / BHIM.
    3. 
          
  3. Tune Approve retries, Restart threshold and Re-login cycles when an IP/edge keeps rejecting — see each field's tooltip for guidance.
    4. 
        

        
⚠️ India region needs residential proxies for production — datacenter IPs get blocked by Turnstile aggressively.

      


      

        
🛟 Troubleshooting

        

          
Captcha / Turnstile keeps failing
Switch to a residential proxy (especially for India). Datacenter IPs are rate-limited and burned quickly.

          
HTTP 409 invalid_state on password verify
Switch Login flow to anti409 in Settings → Proxies.

          
UPI confirm returns blocked / 403 / 429
Increase Re-login cycles (try 2–3) and set Block-streak to break the cycle quickly. Rotate to fresh IPs.

          
OTP never arrives
Verify the mail mode credentials. For iCloud v3, the api_url must be a valid Worker v2 readmail URL.

          
UI shows blank / 401
URL must include ?token=<your_token>, or set X-API-Token header. Token is in .env as GPT_SIGNUP_WEB_TOKEN.
